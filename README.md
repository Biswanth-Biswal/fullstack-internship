# Fullstack Internship

This repository contains tasks and projects completed during my Full Stack Web Development Internship. The goal of this internship is to learn and implement backend and frontend technologies by building practical applications.

## Internship Tasks

### Level 1 – Basic

* Development environment setup (Node.js, npm, Git, GitHub)
* Build a simple REST API using Node.js and Express
* Create a frontend using HTML, CSS, and JavaScript

### Level 2 – Intermediate

* Build frontend using a JavaScript framework (React / Vue / Angular)
* Implement authentication and authorization
* Integrate a database (MongoDB / MySQL / PostgreSQL)

### Level 3 – Advanced

* Develop a complete full-stack application
* Implement real-time features using WebSockets
* Build APIs using GraphQL

## Technologies Used

* Node.js
* Express.js
* HTML
* CSS
* JavaScript
* Git & GitHub
